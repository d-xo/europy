# Model Based Testing

- Model system as a graph (finite state machine)
- Generate tests by randomly walking over graph 
<br>
- Tests are more *flexible*
- Tests find *new bugs*

---

## Building the Graph

- Nodes are states to be verified
- Edges are actions to be taken
